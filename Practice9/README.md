Practice 9 — Pygame Game Development
Three classic mini-applications built with Pygame.

Projects
Folder	Description
mickeys_clock/	Analog clock using Mickey Mouse hand images
music_player/	Interactive keyboard-controlled music player
moving_ball/	Arrow-key driven ball with boundary checking
Quick Start
pip install -r requirements.txt

# Run any project
python mickeys_clock/main.py
python music_player/main.py
python moving_ball/main.py
Requirements
Python 3.8+
pygame >= 2.0.0
